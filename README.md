# Coffee_Shop_Sales
Analyzed coffee shop sales data using Microsoft Excel (Pivot Tables, Charts) to identify revenue trends, top-selling products, and peak business hours.
Performed data cleaning and transformation using Excel functions (IF, VLOOKUP/XLOOKUP, TEXT, DATE) to ensure accurate and structured analysis.
Built interactive dashboards with slicers and visualizations to generate actionable insights for improving sales performance and decision-making.
